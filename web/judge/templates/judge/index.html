How to submit?

  1. Register an account at GitHub.
  2. Create a new private repository for this course.
  3. Add the account{{ github_account }} as a collaborator in your repository Settings -> Manage access.
  4. Set up the GitHub settings in your profile.
  5. Clone the repository into whatever place you like (git clone).
  6. In the directory you just cloned, create another directory named {{ sample_problem.pk }}, which is the problem ID of {{ sample_problem.title }}.
  7. Add all required files marked submitted in {{ sample_problem.title }} into {{ sample_problem.pk }} (with your solutions included, of course).
  8. Commit and push (git add, git commit, and then git push).
  9. Click the submit button in {{ sample_problem.title }}.
  10. See if there are any errors; if so, then go back to debug and try it again when you are done.
Here, we use {{ sample_problem.title }} as sample. Do not forget to make corresponding changes when you are solving other problems.
